Character reference

That new sentence you added has a problematic character at the beginning of it: <. You might notice that once you put < into Visual Studio Code, things look weird when you press Save. Why? A less-than character (<) confuses Visual Studio Code because it thinks you’re starting an HTML tag that doesn’t really exist. The way to fix this problem is by changing the less-than symbol into a character reference, so don't forget to do that.
