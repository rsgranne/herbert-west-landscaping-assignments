Notes on Coding

Spaces

Never use spaces in webpage folder or file names!

This is wrong: landscape maintenance.html

These are right:

  • landscapemaintenance.html
  • landscapeMaintenance.html
  • landscape_maintenance.html
  • landscape-maintenance.html ← what most developers use, & what we will use

Capitalization

Never capitalize webpage folder or file names!

Don’t do this: About-Us/Contact-Info.html

Do this instead: about-us/contact-info.html

Why?

  • You won’t have to remember when you capitalized & when you didn’t
  • Consistency
  • Almost everyone else in the world does it that way
