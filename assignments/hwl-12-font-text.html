In this assignment you will use these CSS properties:

  • line-height: Specifies the vertical space between lines of text within a block-level element. The value should be a number without units, like 1.3. 1.3 would mean 1.3 times the total height of the font, which is measured from the bottom descender to the top ascender.
  • text-transform: Specifies capitalization of text. There are 4 values: capitalize (Every Word), uppercase (ALL CAPS), lowercase, & none (do not change the current capitalization).
