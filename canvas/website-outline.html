We are going to build a website during this course for a Canadian lawn care company named Herbert West Landscaping. Before we start, I thought it would be helpful to see an outline of the entire site.

A section overview is a page that introduces an area of a website. index.html is almost always the name of those files.

Here’s the key for the following outlines:

📄: page
📂: folder
📝: section overview

When you open the herbert-west-landscaping folder that contains the website, you are going to see a home page (index.html) & six folders: about, contact, products, projects, services, & tips (we never use capital letters in folder or file names).

Inside each of those folders, you will find a section overview page named index.html & often other webpages appropriate for that section. For instance, the products folder contains index.html, fertilizers.html, pesticides.html, & seed.html, but the tips folder contains only index.html.

Therefore, the website will have 6 folders:
